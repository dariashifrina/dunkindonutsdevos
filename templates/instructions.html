Instructions and More!


Welcome to New York City Food Inspection Violations by DunkinDonutsDevos. Using the d3 library, we have created a map that displays the density and frequency of health code violations by food establishments in a given zipcode..

Based on the number of violations commited, the area on the map defined for that zipcode will be colored in a certain shade of orange. The stronger the opacity of orange in that area, the higher the number of violations, and vice versa.

Using the slider, you can scroll through different years in order to display the pattern of food violations in different zipcodes. If you want to see the number of violations over all four years, you can check the box titled, "Show Data from All Years".
